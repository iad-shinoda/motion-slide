# 創業と基盤構築 (1999 - 2012)
## 1999年
- 有限会社として創業。
- データ入力・IT系テレセールス(成功報酬型)サービス開始
## 2004年 - 2009年
- 「Dr.Marketing」提供開始・クラウド化。P-Mark取得、商標登録
## 2010年 - 2012年
- BtoBスコアリング機能で特許取得。
- テレセールス実績2000件突破。「電話道」商標登録

# デジタル革新と新たな提携 (2020 - 2022)
## 2020年
- 中小企業デジタル化応援隊事業パートナー就任
- 米国Freshworks社 販売代理店契約

# デジタル革新と新たな提携 (2020 - 2024)
## 2023年
- インド、ドイツ、イスラエル、シンガポールのグローバルIT企業と取引・代理店契約を拡大
- IoTサイトは記事数日本一を達成
- AI問合せツール「Dr.AI® 」及び「Quickwork Automation」をリリース
## 2024年
- 台湾のWishing SoftwareとEHS管理ツールの代理店契約を締結し、IT-Weekで紹介開始
- IT導入補助金支援事業者に認定。
